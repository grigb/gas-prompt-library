# P-009: Prepare Owner-Blocked WOs for Instant Closure

**Action:** Create ready-to-run scripts in `.dev/ai/scripts-temp/` and owner action cards. When the owner acts, closure should be zero-effort.
**Why:** Owner's time is the scarcest resource. Minimize what they need to do.
**Examples:** WO blocked on API key setup -- create a script that configures everything once the owner provides the key.
