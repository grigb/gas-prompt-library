# P-011: Session Handoff at End

**Action:** Create comprehensive handoff at `.dev/ai/handoffs/` with: what was done, what's remaining, evidence links, statistics, priority next steps.
**Why:** Every session ends. The next session (or agent) needs full context.
**Examples:** Context limit approaching -- create handoff with 12 completed WOs, 3 remaining, deploy evidence links.
