# General Prompts

This directory is for reusable, project-agnostic prompts.

Do not place project-bound directives here.

Project-specific K2B directives belong in:
- `~/.agents-gas-prompt-library/handoffs/k2b-project-directives/`

## Available General Prompts

- `project-knowledge-vault-discovery-pass.md` - Ask a project orchestrator or
  worker to search owner knowledge vaults and project `.dev/ai` records for
  forgotten or adjacent knowledge that may add, change, simplify, defer, or
  remove project features.
